## 📌 Overview

This repository contains **Data Structures and Algorithms practical codes** as per the **SPPU 2024 curriculum**, implementing core concepts like linked lists, trees, graphs, searching, and sorting.

### Key Focus Areas

| Area | Topics |
|------|--------|
| **Searching & Sorting** | Linear/Binary Search, Bubble, Selection, Insertion Sort |
| **Linked Lists** | Singly Linked List (SLL), Doubly Linked List (DLL) |
| **Stacks & Queues** | LIFO/FIFO operations, expression parsing |
| **Non-Linear DS** | Binary Trees, Binary Search Trees, Graphs (BFS/DFS) |
| **Hashing** | Hash functions, collision resolution techniques |

## 🚀 How to Use

1. **Navigate to the Data Structures folder**:
```bash
cd Data_Structures
```

2. **Run Python Practicals**:
```bash
python Practicals/Practical_01A.py
```

3. **Compile and Run C Practicals**:
```bash
gcc ENTC_DS_Practicals/Practical_01.c -o p1
./p1
```
