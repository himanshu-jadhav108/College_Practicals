## 📌 Overview

This repository contains **Data Science practical codes** as per the **SPPU 2024 curriculum**, implementing machine learning algorithms, data analysis, and visualization techniques.

### Key Focus Areas

| Area | Topics |
|------|--------|
| **Data Preprocessing** | Cleaning, scaling, normalization, handling missing values |
| **Statistical Analysis** | Probability, Linear Algebra, EDA |
| **Machine Learning** | Linear Regression, Logistic Regression, Classification |
| **Data Visualization** | Matplotlib, Seaborn, interactive plots |
| **Business Intelligence** | Power BI Dashboards, DAX, Power Query |

## 🚀 How to Use

1. **Navigate to the Data Science folder**:
```bash
cd Data_Science
```

2. **Install requirements**:
```bash
pip install -r requirements.txt
```

3. **Run Python Scripts / Notebooks**:
```bash
# Python scripts
python Practical_05/Basic/practical_05_basic.ipynb

# Jupyter Notebooks
jupyter notebook Practical_05/Advance/practical_05_advance.ipynb
```
