# Data Science Practicals – SPPU 2024

This folder contains **Data Science practical codes** implementing machine learning algorithms, data analysis, and visualization techniques as per **SPPU 2024 pattern**.

---

## 📌 Overview

This repository focuses on:
- **Data Preprocessing & Cleaning** – Handling missing values, normalization, and feature engineering
- **Exploratory Data Analysis (EDA)** – Statistical analysis and data visualization
- **Machine Learning Models** – Supervised and unsupervised learning algorithms
- **Data Visualization** – Creating insightful visual representations using matplotlib, seaborn, etc.

## 🚀 How to Use

1. **Navigate to the Data_Science folder**:
```bash
cd Data_Science
```

2. **Install required packages** (if not already installed):
```bash
pip install -r requirements.txt
```

3. **Run any practical**:
```bash
python practical_name.py
```

or for Jupyter notebooks:
```bash
jupyter notebook notebook_name.ipynb
```
