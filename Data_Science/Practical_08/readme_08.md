# 👨‍💼 HR Analytics Dashboard (Power BI)

## 🧪 Experiment No: 08

## 📌 Title: HR Analytics Reporting using Power BI

---

## 📖 Overview

This project focuses on analyzing employee data to understand workforce trends and attrition patterns using Power BI.
The dashboard helps HR teams make data-driven decisions.

---

## 📂 Dataset

* **Source:** Kaggle
* **Dataset:** IBM HR Analytics Attrition Dataset
* **Path:** `Dataset/IBM_HR_data.csv`
* **Link:** https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset

---

## 🎯 Objectives

* Analyze employee distribution across departments
* Measure attrition rate
* Understand workforce demographics
* Identify patterns affecting employee retention

---

## 📊 Dashboard Features

* 👥 Headcount by Department
* ⚖️ Gender Diversity (Pie Chart)
* 📊 Average Tenure by Department
* 📉 Attrition Rate (KPI Card)
* 🎂 Age Distribution (Histogram)

---

## 🧮 Key Metric

**Attrition Rate = Employees Left / Total Employees**

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query
* DAX (Calculated Measures)

---

## 🧠 Key Insights

* Identified departments with high attrition rates
* Observed gender distribution across workforce
* Analyzed tenure trends impacting retention
