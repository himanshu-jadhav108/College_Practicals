## 📌 Overview

This repository contains **Object-Oriented Programming (OOP) practical codes** as per the **SPPU 2024 curriculum**, focusing on core concepts using **Java** to solve complex problems through modular design.

### Key Focus Areas

| Area | Topics |
|------|--------|
| **Fundamental Concepts** | Classes, Objects, Constructors, Methods |
| **Inheritance & Polymorphism** | Method Overloading, Method Overriding, Single Inheritance |
| **Interfaces & Abstraction** | Defining contracts using interfaces and abstract classes |
| **Error Handling** | Exception Handling (Try-Catch-Finally) |
| **Real-world Modeling** | E-commerce systems, ATM simulations, Calculator logic |

## 🚀 How to Use

1. **Navigate to the OOP folder**:
```bash
cd Object_Oriented_Programming
```

2. **Compile the Java Practical**:
```bash
# Example: Practical 1
javac practical_01/Calculator.java
```

3. **Run the compiled class**:
```bash
# Important: Use the package name notation
java practical_01.Calculator
```
