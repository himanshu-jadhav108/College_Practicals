## **Experiments Overview**

### **Basic Experiments**
1. Expert System ‚Äì Rule-based decision making.
2. BFS & DFS Maze Solver ‚Äì Graph search algorithms.
3. A* Algorithm Maze Solver ‚Äì Heuristic-based pathfinding.
4. CSP Map Coloring ‚Äì Backtracking solution for map coloring.
5. Minimax Tic-Tac-Toe ‚Äì Basic AI decision making.
6. Minimax with Alpha-Beta Pruning ‚Äì Optimized AI decision making.

> **Folder:** `Aritificial_Intelligence/Basic/`  

---

### **Advanced Experiments**
1. Expert System ‚Äì Interactive and visual interface.
2. BFS & DFS Maze Solver ‚Äì Animated step-by-step exploration.
3. A* Algorithm Maze Solver ‚Äì Animated pathfinding visualization.
4. CSP Map Coloring ‚Äì Real map coloring with interactive display.
5. Minimax Tic-Tac-Toe ‚Äì Animated moves with AI evaluation scores.
6. Minimax with Alpha-Beta Pruning ‚Äì Animated AI moves with pruning visualizations.


