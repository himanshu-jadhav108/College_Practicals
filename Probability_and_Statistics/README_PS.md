## 📌 Overview

This repository contains **Probability and Statistics practical codes** as per the **SPPU 2024 curriculum**. It focuses on data distribution analysis, correlation studies, and real-world case studies using Python's statistical libraries.

### Key Focus Areas

| Area | Topics |
|------|--------|
| **Data Distribution** | Visualizing distributions, Traffic flow analysis |
| **Correlation Analysis** | Pearson correlation, Salary vs Experience studies |
| **Statistical Visualization** | Histograms, Scatter plots, Regression lines |
| **Exploratory Data Analysis** | Insights from real-world datasets |

## 🚀 How to Use

1. **Navigate to the Probability & Statistics folder**:
```bash
cd Probability_and_Statistics
```

2. **Install requirements** (if not already installed):
```bash
pip install -r ../requirements.txt
```

3. **Launch Jupyter Notebooks**:
```bash
jupyter notebook Case_Study_Unit_3/traffic_analysis_case_study.ipynb
```
